# To-Do List Application

## Overview

```
The To-Do List Application is a user-friendly web application that allows users to manage their tasks efficiently. Users can add new tasks, update existing ones, and remove tasks they no longer need. This application helps in organizing daily activities and enhancing productivity.
```

## Technologies Used

```
Vue.js: A progressive JavaScript framework for building user interfaces.

Vuex: A state management pattern + library for Vue.js applications.

Bootstrap: A front-end framework for developing responsive and mobile-first websites.

Local Storage: For persisting task data in the user's browser.
``` 

## Primary Functions

```
Add To-Do: Users can create new tasks by entering a title, description, and status (low, medium, high). Tasks can be added individually or in bulk (up to 100 at a time).

Update To-Do: Users can edit existing tasks to modify their title, description, or status. This allows for easy management of tasks as priorities and details change.

Remove To-Do: Users can delete tasks that are no longer needed, helping to keep their to-do list clean and relevant.
```
